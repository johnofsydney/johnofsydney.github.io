Why is Pipeline free?


Pipeline

I made Pipeline to run my own small recruitment business. It developed into a pretty comprehensive tool which I licensed to several other small recruitment agencies. But I'm no longer actively developing further features - it will never run on a mac or iphone / android, it will never get candidates directly from linked in or post to multiple job boards. So I have decided to make it freeware. If you want customisation or training please send me an email to discuss.
