Reports



Several reports are available, listed in the list box on the left. Double-click the report name to launch the report. For reports where the data is restricted by date, the data is that between the Start Date and End date fields on the right. By Default Start Date is the first day of the current year, and End Date is Today. Both can be edited as required.
