Search

This screen, used in conjunction with the Mailshot Screen is perhaps the most powerful and useful part of Pipeline. Please take the time to understand how to use this screen

To start with this screen opens with a list of ALL contacts on the database so long as

  • The contact has any text (such as a keyword) in the Searchable Notes and Keywords field AND
  • The Exclude From Searches check box on the Contact screen is NOT ticked

    • Keyword Filtering
    Fifteen search terms are available, in a combination of AND and OR. Below the button marked Search LinkedIn you can see a plain English description of what you are searching for. Name Filtering
    Name example: Entering “john” in the Name field will find all contacts with john in any part of their name. “John Smith” and “Al Johnson” will both be found. The same filtering principles apply to all of the other fields shown in the filter section. The Email filter box will search in both Personal and Work Email addresses The Mobile filter box will search Work, Home and both Mobile phone numbers Action Buttons

  • Export to Excel: Send your filtered list to Excel for additional work
  • Mailshots: Send your filtered list to the Mailshot Screen.
  • Deep Search: sends the search terms to Windows Search which can search in emails, and in the text of CVs and other documents. Settings for the Windows Search index can be found in your Control Panel.
  • Search LinkedIn: Sends the search terms to Google to search the LinkedIn site for those terms.
