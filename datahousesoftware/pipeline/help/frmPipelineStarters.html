Starters

There are several screens available under the Pipeline button in the Pipeline toolbar. The purpose of all of these Pipeline screens is to illustrate the sales funnel or Pipeline, from whence the software Pipeline takes it's name. These are designed as both a tool for both management and individual consultants.

The basic premise is that a candidate will enter the sales funnel when his or her CV is sent to a client, and will exit when they are rejected or start their new job.

CV Sent >> Interview Stage >> Offer >> Forward >> Starter

USe these screens to show which clients need to provide feedback, and which candidates need to receive it.

This screen shows all of the candidates who have started a job between the date range shown on the right. By default the date range covers the last 90 days, but it can be adjusted to show any date range you wish.

To show on this screen the candidate must be recorded as having started the job, which means on the job screen

  • the Start text box must be completed
  • the date in the Start text box must have been reached, and
  • the Started check box must be ticked
