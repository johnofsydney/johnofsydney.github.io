Importing Data into Pipeline

  1. Download ZIP file which contains the blank excel spreadsheet; tblContact.xlsx
  2. Enter your data into the spreadsheet.
  3. Leave the headings exactly as they are. Do not delete the headings, not change the names / spelling or order of the columns.

Backup your data before running import procedure


  • ContactForename: Forename
  • ContactSurname: Family Name
  • MiddleName: Middle Name. Leave blank if none.
  • PreferredName: Preferred Name. Leave blank if none.
  • ContactJobTitle: Job Title (most applicable for Client Contacts). Leave blank if not known.
  • ContactOfficePhone: Landline Work Phone. Leave blank if not known.
  • ContactHomePhone: Landline Home Phone. Leave blank if not known.
  • ContactWorkEmail: Work Email Address. Leave blank if not known.
  • ContactHomeEmail: Home / Personal Email Address. Leave blank if not known.
  • ContactMobile: Work Mobile / Cell Phone number. Leave blank if not known.
  • PersonalMobile: Personal Mobile / Cell Phone Number. Leave blank if not known.
  • ContactWorkAddress: Work / Office Address. Leave blank if not known.
  • ContactWorkCity: Work / Office Town / City. Leave blank if not known.
  • ContactWorkState: Work / Office State or County. Leave blank if not known.
  • ContactWorkPostcode: Work / Office Postcode. Leave blank if not known.
  • ContactHomeAddress: Home Address. Leave blank if not known.
  • ContactHomeCity: Home Town / City. Leave blank if not known.
  • ContactHomeState: Home State / County. Leave blank if not known.
  • ContactHomePostcode: Home Postcode. Leave blank if not known.
  • ContactGeneralNotes: Enter any KEYWORDS here. Such as skills / qualifications / etc. Enter company name here.
  • Source: Where did this contact come from (eg Seek / My Career / Client / referral / etc) Used for reporting.
  • DateAdded: Enter today's date in format dd/mm/yyyy (eg 31/12/2013) *
  • Consultant: username of Pipeline consultant to whom this contact will be tagged.
  • ResumeFilenamePath: Full path to original candidate CV eg "J:\Resumes\Original\Fred Flintstone.doc"
  • ResumeForClients: Full path to candidates CV that you will email to clients eg "J:\Resumes\Formatted\Fred Flintstone.doc"
  • MailingList: Enter "TRUE" (wihout quotes) to allow this contact to appear on Mailshot screen. If left blank will default to FALSE. For most contacts you should enter TRUE
  • DateOfBirth: Enter Date of Birth if known in format dd/mm/yyyy
  • ContactFreeNotes: Enter here any other notes for contact. These are NOT searchable by Pipeline
  • OtherDocuments: Full path to any other documentation
  • InterviewNotes: Full path to any other documentation
  • ExcludeFromSearches: Enter "TRUE" to prevent this contact from appearing in searches. If left blank will default to FALSE
Once your data has been succesfully squeezed into the format listed above
  1. Click on Consultancy in toolbar
  2. Enter the password "pivotal" (if this has been changed on your database, use the new password) and press enter
  3. Click on the button Edit Users and Consultancy
  4. Click on the tab marked Actions
  5. On the right hand side click on the button marked Import Contacts
  6. Browse to find the spreadsheet with your data, select it and the import will begin.
  7. A notification will shown when the import has finished.
